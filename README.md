## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/Wasif-Sohail55/ElectricityCalculator.git

# Navigate to directory
cd ElectricityCalculator

# Run the application
java -jar ElectricityCalculator.jar
```

### Requirements
- Java 17+
- 100MB Disk Space
- 1024×768 Resolution
